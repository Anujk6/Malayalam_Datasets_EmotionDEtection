# MalEmo & EmoVec-Mal: Malayalam Emotion Detection Datasets

## MalEmo
A six-class Malayalam emotion detection dataset of 15,749 sentences,
created by translating the Kaggle Emotion Recognition Dataset into
Malayalam with manual verification (Krippendorff's Alpha = 0.86).

### Emotion Categories
| Category | Train Count |
|----------|-------------|
| Happy    | 4,764       |
| Sadness  | 3,988       |
| Fear     | 1,380       |
| Love     | 1,044       |
| Anger    | 914         |
| Surprise | 374         |

### Files
- `MalEmo/train.csv` — 12,464 samples
- `MalEmo/test.csv`  — 1,677 samples
- `MalEmo/val.csv`   — 1,608 samples
